# O Jogo
Leo é um jovem bibliotecário que vê sua rotina pacata ser interrompida quando é sugado para o mundo de Elin. Armado com a Lanterna de Éter, Alex usa a luz para materializar pontes e objetos em meio às ruínas do mundo. O jogo se passa em uma biblioteca mágica corrompida, onde todo o conhecimento está sendo destruído. Elin é a dimensão onde vivem todas as histórias já escritas — mas esse mundo está morrendo. O jogador deve eXplorar a fase me busca das páginas perdidas de livros, para conseguir o poder capaz de derrotar o viLão responsável pela destruição de Elin. Só assim Leo poderá trazer paz para este mundo e libertar seu povo.
